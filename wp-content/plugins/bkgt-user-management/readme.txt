Användarhantering och laghantering för BKGTS Ledarsystem.

== Description ==

Ett specialanpassat plugin för hantering av användare, roller och laguppdelningar för BKGTS American Football.

Funktioner:
* Laghantering (Teams)
* Användarroller: Styrelsemedlem, Tränare, Lagledare
* Laguppdelningar och tilldelningar
* Behörighetshantering
* Team-baserad åtkomstkontroll

== Installation ==

1. Ladda upp plugin-mappen till `/wp-content/plugins/`
2. Aktivera plugin via 'Plugins' menyn i WordPress
3. Konfigurera lag via Användarhantering > Lag
